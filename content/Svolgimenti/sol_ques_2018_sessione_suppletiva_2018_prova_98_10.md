Si verifica che $y = e^x\sin x$ è soluzione di $y'' - 2y' + 2y = 0$.

**Derivate:**

$$y' = e^x(\sin x + \cos x).$$

$$y'' = e^x(\sin x + \cos x) + e^x(\cos x - \sin x) = 2e^x\cos x.$$

**Sostituzione:**

$$y'' - 2y' + 2y = 2e^x\cos x - 2e^x(\sin x + \cos x) + 2e^x\sin x = e^x[2\cos x - 2\sin x - 2\cos x + 2\sin x] = 0. \quad \checkmark$$
