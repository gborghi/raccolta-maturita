QUESITO $6$

Sono dati un angolo $\alpha$ di $\pi 2$ radianti e un angolo $\beta$ di $539$ gradi. Si verifichi che sono
entrambi maggiori di un angolo giro e minori di due angoli giro. Si dica quale dei due è il
maggiore. Si dica inoltre se e più grande il seno di $\alpha$ o il seno di $\beta$ .

Risulta:

$\alpha = \pi 2 = \pi ∙\pi rad> 2\pi$ rad : quindi $\alpha$ è maggiore di un angolo giro;
$\alpha = \pi 2 = \pi ∙\pi rad< 4\pi$ rad : quindi $\alpha$ è minore di due angolo giro.

$\beta = 539^\circ > 360^\circ$ : quindi $\beta$ è maggiore di un angolo giro;
$\beta = 539^\circ < 720^\circ$ : quindi $\beta$ è minore di due angolo giro;

Per confrontare $\alpha$ e $\beta$ trasformiamo, per esempio, $\alpha$ in gradi:

$\alpha : \pi = a^\circ : 180^\circ \implies \alpha ^\circ = \alpha ∙180^\circ$
$\pi$
$= \pi 2 ∙180^\circ$
$\pi$
$= \pi ∙180^\circ \cong 565^\circ > \beta ^\circ$

Quindi : $\alpha > \beta$ .

$sen(\alpha ) \cong sen(565^\circ ) = sen(565^\circ -360^\circ ) = sen(205^\circ ) = sen(180^\circ + 25^\circ ) = -sen(25^\circ ) \cong -0.423$

$sen(\beta ) \cong sen(539^\circ ) = sen(539^\circ -360^\circ ) = sen(179^\circ ) = sen(180^\circ -1^\circ ) = sen(1^\circ ) \cong 0.017$

Quindi : $sen(\alpha ) < sen(\beta$ ) .
