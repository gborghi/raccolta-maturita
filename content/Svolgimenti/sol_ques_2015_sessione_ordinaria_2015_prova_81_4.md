$SD2 = SO2 -OD2 = t^{2} -4$

Pertanto:

$t^{2} -2t+ 4 = t^{2} -4$ , da cui; $t= 4.$

Il punto $S$ richiesto dista quindi $4$ dal centro della sfera.

N.B.

Osservato che il triangolo SDO è rettangolo in $D$ e che $AD$ è l’altezza relativa
all’ipotenusa, applicando il secondo teorema di Euclide si ottiene più velocemente $t.$
Si ha infatti: $DA2 =$ AO∙SA , $3 = 1$ ∙SA , $SA= 3 , t-1 = 3, t= 4$ .

QUESITO $4$

Sia P(x) $= x^{2} + bx+ c.$ Si suppone che $P(P(1)) = P(P(2)) = 0$ e che $P(1) \ne P(2).$
Calcolare $P(0).$

Risulta: $P(1) = 1 + b+ c , P(P(1)) = (1 + b+ c )^{2} + b(1 + b+ c ) + c$

$P(2) = 4 + 2b+ c , P(P(2)) = (4 + 2b+ c )^{2} + b(4 + 2b+ c ) + c$

$P(0) = c$

$1 + b+ c\ne 4 + 2b+ c \implies b\ne -3$

$(1 + b+ c )^{2} + b(1 + b+ c ) + c= 0$
$(4 + 2b+ c )^{2} + b(4 + 2b+ c ) + c= 0$
$b\ne -3$
$2b^{2} + 3bc+ 3b+ c^{2} + 3c+ 1 = 0$
$6b^{2} + 5bc+ 20b+ c^{2} + 9c+ 16 = 0$
$b\ne -3$

Sottraiamo alla seconda equazione la prima:

$2b^{2} + 3bc+ 3b+ c^{2} + 3c+ 1 = 0$
$4b^{2} + 2bc+ 17b+ 6c+ 15 = 0 \implies c(2b+ 6) = -4b^{2} -17b-15$
$b\ne -3$

Dalla seconda equazione otteniamo (con $b\ne -3$ , come previsto per ipotesi):

$c= -4b^{2} -17b-15$
$2(b+ 3)$
$= -(b + 3)(4 b + 5)$
$2(b+ 3)$
$= -4b+ 5$
$= P(0)$
